# HOW TO USE THIS WEBSITE

Welcome to your new portfolio, Emerson! This website was designed to be simple, cute, and ready to go.

## 📂 Project Structure
- `index.html`: Your main homepage.
- `about.html`: Facts about your school, major, and goals.
- `hobbies.html`: A section dedicated to your creative interests.
- `style.css`: All the colors, fonts, and Y2K styling.
- `script.js`: Handles the Dark Mode toggle and animations.
- `assets/`: This is where you should keep your images and PDF files.

## 📸 Profile Photo
To change your photo:
1. Save your photo in the `assets/` folder and name it (e.g., `profile.jpg`).
2. Open `index.html`.
3. Find the `<img>` tag and change the `src` to `assets/profile.jpg`.

## 📄 Resume
To add your resume:
1. Save your resume as a PDF in the `assets/` folder.
2. Name it `resume.pdf`.
3. The button on the Home page is already linked to this file!

## ✍️ Editing Text
- All files have clear comments.
- You can change any text inside the tags (like `<h1>`, `<p>`, or `<h3>`).
- To change colors, open `style.css` and look for the `:root` section at the top. You can swap out the Hex codes (e.g., `#ff85a1`) for any colors you like!

## 🚀 Publishing to GitHub Pages
1. Create a new repository on [GitHub](https://github.com/new).
2. Upload all these files (including the `assets` folder) to that repository.
3. Go to **Settings** > **Pages** in your GitHub repository.
4. Under "Branch", select `main` and click **Save**.
5. Your website will be live at `https://yourusername.github.io/your-repo-name/` within a few minutes!

Enjoy your bubbly new home on the web! ★
